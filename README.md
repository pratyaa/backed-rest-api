### Quickstart
1) Clone the repo
```
 git clone https://github.com/pratyaa/backed-rest-api
 cd backed-rest-api
```

2) Install the dependencies
```
 npm install
```

3) Start the server
```
 npm start
```

4) API endpoint at 

[http://localhost:5000/](http://localhost:5000/)
